# 🍽️ Recipe Finder

A visually appealing and user-friendly recipe discovery website that helps users explore a wide variety of dishes based on their preferences. Whether you're craving something sweet, spicy, or healthy, Recipe Finder offers an organized and engaging browsing experience.

🔗 **Live Website:**  
https://recipe-finder-phi-orpin.vercel.app/index.html

---

## ✨ Features

- 🧭 Clean and modern homepage with hero section  
- 🔍 Search bar UI for exploring recipes  
- 🍴 Categorized sections:
  - Appetizers  
  - Entrée  
  - Desserts  
  - Side Dishes  
  - Mocktails  
  - Meals  
  - Cuisine-based  
  - Diet-friendly recipes  
- 📄 Separate pages for each category  
- 🎨 Visually appealing UI with cards and hover effects  
- 🖼️ High-quality food images and background sections  
- 🎥 Embedded YouTube videos for recipe guidance  

---

## 🛠️ Tech Stack

- **HTML5** – Website structure  
- **CSS3** – Styling and layout  
- **Flexbox** – Alignment and responsive layout  
- **Font Awesome** – Icons  
- **Vercel** – Deployment
